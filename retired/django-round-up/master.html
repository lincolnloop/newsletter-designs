Jeremy Dunck joins the DSF Board

Jeremy Dunck, former Django Software Foundation secretary, was voted into the board seat vacated by Alex Gaynor (who left to become director of the Python Software Foundation). Andy McKay will be stepping in to fill Jeremy Dunck's shoes as DSF secretary.

Interesting Links

Core Concepts of Django ModelForms

Daniel Greenfeld continues his line of informative Django blog posts with this one on what ModelForms do and how to use them.

Making your Django app more pluggable

Some tips and tricks from Dan Poirer at Caktus Group on how to create better reusable apps for Django. Topics include how to handle settings, templates, and dependencies.

Python App Deployment with Native Packages

This one isn't Django specific, but it certainly applies to any Django developers handling deployment. Python core developer, Hynek Schlawack, discusses deploying Python applications as a native OS package (dpkg on Debian/Ubuntu for example). Once the package is properly created, deployment across many servers is trivial and fool-proof.
